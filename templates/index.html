Caveats & roadmap
  • The processor skips the first row, treats the second row as headers, and expects actual data to start on row three. If your export structure differs, the conversion will fail.
  • Only the provided template layout is supported. Protected cells (e.g., Service dropdowns) may flag unfamiliar values in red.
  • Future versions may incorporate an LLM layer—either to locate data blocks and double-check the Python output, or to emit CSV-ready tables end-to-end. The latter is attractive but risky on very large files (LLMs might alter data), so we will only pursue it after collecting solid accuracy metrics.
